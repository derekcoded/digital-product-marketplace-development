## 📖 Project Overview

Digital Product Marketplace is a fully responsive frontend application built using React, TypeScript, Vite, Tailwind CSS, and Zustand.

The platform enables users to browse digital assets, explore product details, manage their shopping cart, complete a checkout workflow, track orders, access support resources, and interact with a modern marketplace experience.

This project demonstrates production-style frontend architecture, reusable components, state management, responsive design principles, and modern React development practices.

## ⚡ Getting Started

### 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/digital-product-marketplace.git
```

### 2️⃣ Navigate to Project

```bash
cd digital-product-marketplace
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4️⃣ Start Development Server

```bash
npm run dev
```

### 5️⃣ Build for Production

```bash
npm run build
```
